				2. Uma coleção de sequências


    Como vocês podem ter notado, várias das operações mencionadas funcionam da mesma forma com textos,
listas e tabelas. Coletivamente, textos, listas e tabelas são chamados de 'trens'.


    Antes de criar o Python, Guido foi um dos desenvolvedores da linguagem ABC—um projeto de pesquisa
de 10 anos para criar um ambiente de programação para iniciantes. A ABC introduziu várias ideias
que hoje consideramos "pythônicas". O Python não é assim tão amigável por acidente.


    O Python herdou da ABC o tratamento uniforme de sequências. Strings, listas, sequências de
bytes, arrays, elementos XML e resultados vindos de bancos de dados compartilham um rico conjunto
de operações comuns, incluindo iteração, fatiamento, ordenação e concatenação.
